# To-Do List App

## Project Overview
The To-Do List App is a modern task management application built with Spring Boot and H2 Database.

## Features
- Create tasks with a title and description
- View all tasks or individual tasks
- Update tasks (title, description, status)
- Delete tasks
- Persistent in-memory database for development

## Technologies Used
- **Backend:** Spring Boot, Spring Data JPA, Hibernate
- **Database:** H2 Database (In-Memory)
- **Build Tool:** Maven
- **Version Control:** Git & GitHub
- **Testing Tools:** Postman, Spring Boot Test

## System Requirements
- **Java Development Kit (JDK):** 17+
- **IDE:** IntelliJ IDEA / Eclipse / VS Code
- **Maven:** Latest Version
- **Git:** Latest Version
